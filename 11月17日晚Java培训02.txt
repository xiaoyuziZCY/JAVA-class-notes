1、函数的定义：用于实现某种特殊功能的一段代码，可以被重复调用，提高代码使用效率。
2、函数的定义格式：
	public void method(int a){
		System.out.Println(a);
	}

	//
	函数修饰符  函数的返回值类型 函数名称(函数的参数){
	   //函数体
	}

	注意点：
		① 修饰符：public、private、protected、
		② 返回值类型：单返回值类型
		③ 函数的重载
		④ 特殊修饰符：static、final 

3、函数的调用：
	① 非静态函数：只能被类的实例所拥有，必须得new对象，然后调用
	② static修饰：可以在main中直接调用使用.

4、参数的分类：
	形参：函数在定义或者声明的时候，所定义的称之为形参
	实参：实际参数，在真正调用函数的时候，所传递的参数，称之为实参。

5、返回值和返回值类型：
	返回值类型：在函数定义或者声明的行写的是返回值类型，返回值类型是一个数据类型，比如说int、String、boolean
	返回值：在函数体内，return语句之后跟着的是返回值, 指的是一个具体的变量，或者一个具体的数值

6、函数的开发原则：独立功能原则，一个函数只做一件事
	逻辑思维能力.

7、数组：本质是一个容器。

8、数组：在计算机中，一组连续的内存空间，作为数组的存储位置。
  
9、数组的特点：可以通过数组的下标访问任意的数组元素，下标从0开始。

10、数组的创建：
	int[] a = new int[5];

	元素的访问：a[index]，index从0开始，到length
	数组的长度：a.length;

	注意：ArrayIndexOutofBoundsException,数组下标越界异常

11、数组的扩容和复制：
	① 开发者手动编写：手动创建新数组并遍历赋值
	② Java的API：
		System.arraycopy();
		Arrays.copyOf();

13、多维数组：二维及以上的数组称之为多维数组
    一维数组：一行， 例如{1,3,5,7,9}
    二维数组：平面矩阵
    三位数组：三维立体
    ...

14、面向对象：OOP
	任何的类都是基于一个叫Object的基类。
	对象公有两组类别组成：一类是特征，名词用于描述事物本身的属性；还有一类是行为，动词用于描述事物的功能。

15、方法的重载: 同一个类中，同名的方法称之为重载.
	重：重复

	同一个类中，相同方法名的方法具备不同的参数个数，参数类型，参数顺序等，这种称之为方法的重载.

16、面向对象的三大特征：封装、继承、多台。
	
17、继承:用于描述类与类之间的关系,主要用于类之间的属性和方法的共享.

	Java: 继承是一种单继承. 一个类只能有一个父类。

	语法：extends
