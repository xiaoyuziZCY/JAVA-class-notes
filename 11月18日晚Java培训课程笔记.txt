11月18日晚Java培训课程笔记

1、多态：父类的引用指向之类的对象.
	本质: =号左边范围大，=号右边范围小。
	经验：左边是父类，右边是子类
2、注意点：
	① 向上转型: 上指的是父类一级，可以自动转
	② 向下转型：下指的是子类一级，手动强制转换
	③ 异常：ClassCastException
	④ 关键字：instanceof，返回值是一个boolean类型
3、修饰符：
	private、protected、default、public、static、abstract

4、抽象的：抽象类、抽象方法
注意点：
	① 如果方法是抽象的，类一定是抽象的，即一定要有abstract
	② 类如果是抽象的，方法没有显示，可以是抽象的，也可以不是抽象的。
	③被abstract修饰的类，不能new对象实例。
	④ 抽象类可以被继承，
5、static：静态
① static可以修饰：字段、方法、类、代码块
	静态：在内存当中单独开辟一份控件，用于存放静态的内容，所有的实例共享同一份内存空间。
② 非静态方法中可以直接访问静态的属性
   静态方法中不可以直接访问非静态的属性

6、静态代码块和动态代码块：
static {
	todo...
}

动态代码块：与new结合使用。即只有通过new创建对象的时候，才会触发动态代码块的执行:
{
	todo...
}

优先级：静态代码块 > 动态代码块 > 构造方法

7、final：最后的，最终的
final可以修饰：属性，方法，类
① 修饰属性: 属性值不能改变, 即常量.
② 修饰方法：方法不能被重写,但可以被继承
③ 修饰类：不能被继承

8、接口：interface

9、常用类：
	object类常用方法：
	 	equals:默认比较的地址,对象实例的地址.如果想要按照自己的逻辑进行比较，则可以在类中重写equals方法.
